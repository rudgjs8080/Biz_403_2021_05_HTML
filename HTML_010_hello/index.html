block형 tag

block 형 tag는 Content의 양과는 관계없이 무조건 한 라인을 차지한다

block 형 tag는 한라인에 2개의 tag를 배치하기 어렵다

inline 형 tag

inline 형 tag는 Content의 양만큼만 영역을 차지한다

Content의 양이 적으면 한라인에 연속 배치된다

a, span, label 등등이 있다

block 형과 inline 형은 필요에 따라 변환할 수 있다
